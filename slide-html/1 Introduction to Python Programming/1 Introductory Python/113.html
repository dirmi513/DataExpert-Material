Print Function

For the previous slide's coding challenge, the command that you had to execute was print(20 * 3 + 150). This command utilized Pythonâ€™s built-in print() function (we will explore functions in great depth later on). What happened behind the scenes when you executed print(20 * 3 + 150)?

  1. Python first performed the calculations within the parenthesis, 20 * 3 + 150. The order of operations in Python is determined by PEMDAS:
    • First, 20 was multiplied by 3 to get 60.
    • Then, 150 was added that product of 60 to get the final value of 210.
  2. print(210) was executed, which instructs Python to display the value that is within the parenthesis of print(210) on your screen.

If you had just entered 20 * 3 + 150 into the text editor without using the print() function, Python would still perform the calculations behind the scenes, but result would not be displayed on the screen. That is why we use the print() function.

  • There are multiple, distinct arithmetic expressions, and each one is contained in a separate print() function.
  • Each time the print() function is used, it is on a new line in the source code editor. Please note that it does not matter if the next print() function is on the line directly below the current one, or five lines down, as long as it is on a new line.
  • The output of executing the source code displays the resulting value of each expression in the same order as the commands are written in the source code. Python executes your source code from top to bottom.
