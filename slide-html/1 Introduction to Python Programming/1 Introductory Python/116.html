Key Concepts

  • Python is one of the most, if not the most, widely used programming languages. This is due to its versatility, ease of use, and continued support for almost all use cases.
  • Humans utilize programming languages to instruct the computer to perform specific commands, the most basic being arithmetic calculations.
    • To perform basic arithmetic in Python, you need to use its arithmetic operators: + - * / **.
  • The print() function can be used to display numeric data, such as integers and floats. It can also be utilized do display other types of data that is non-numeric, which we will learn about in the next lesson.
  • Just like natural, human languages have grammar rules that define the structure of the language, programming languages have syntax rules. If you have incorrect syntax in your Python source code, i.e. SyntaxError, Python will not execute your code, but it will point you to the source of the error.
  • We can make use of comments in our source code to make it easier to understand.
